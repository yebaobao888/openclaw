# AI 工具使用指南

## 🎯 核心工具：浏览器

你的主要工作工具是 **browser**，通过浏览器可以访问各种 AI 工具来完成任务。

## 🛠️ 可用的 AI 工具

### 1. ChatGPT (https://chatgpt.com)
**用途**: 通用 AI 对话、内容生成、代码编写
**使用场景**:
- 生成内容草稿
- 头脑风暴创意
- 分析数据和趋势
- 编写脚本和代码

**使用方法**:
```
1. browser 工具访问 https://chatgpt.com
2. 在输入框输入提示词
3. 等待回复
4. 复制结果
```

### 2. Gemini (https://gemini.google.com)

## 🎯 工具选择策略

### 调研阶段
**首选**: Perplexity（带引用，可靠性高）
**备选**: Gemini（实时信息）、Grok（社交媒体趋势）

**示例任务**: 调研 AI 工具变现策略
```
1. Perplexity 搜索 "AI tools monetization strategies 2026"
2. Grok 搜索 "AI creator income twitter"
3. Gemini 搜索 "AI content creator revenue models"
```

### 内容创作阶段
**首选**: ChatGPT（内容生成）、Manus（长文写作）
**备选**: Gemini（多语言内容）

**示例任务**: 创建 X 帖子
```
1. ChatGPT 生成 10 个帖子草稿
2. Manus 整理成内容库
3. 选择最佳版本
```

### 视觉内容阶段
**首选**: Leonardo.ai（图像）
**备选**: Suno（音频，如果需要）

**示例任务**: 创建 YouTube 封面
```
1. Leonardo.ai 生成封面图
2. 下载并保存
```

### 趋势分析阶段
**首选**: Grok（X 平台）、Perplexity（全网）
**备选**: Gemini（Google 趋势）

**示例任务**: 分析当前热点
```
1. Grok 查看 X 平台热门话题
2. Perplexity 深度分析话题背景
3. Gemini 查看搜索趋势
```

## 🔄 工具组合使用

### 完整工作流示例：创建一篇 AI 工具评测

```
第 1 步：调研（Perplexity）
- 搜索 "best AI tools for content creators 2026"
- 收集 10 个工具信息

第 2 步：深度分析（ChatGPT + Gemini）
- ChatGPT 分析每个工具的优缺点
- Gemini 查找用户评价和案例

第 3 步：内容创作（Manus）
- 在 Manus 中整理研究结果
- 撰写评测文章

第 4 步：视觉素材（Leonardo.ai）
- 为每个工具生成配图
- 创建对比图表

第 5 步：社交媒体版本（ChatGPT）
- 生成 X 帖子版本（280 字）
- 生成 YouTube 脚本版本
- 生成小红书版本
```
